# react-trivia
Trivia Game using React-app
By: Crissy Pohl

Requirements:
1. At least 20 questions and answers
2. 2 or more components
3. 1 or more buttons for interactivity
4. Show one question or answer at a time
5. Move forward or back through the trivia questions
6. When moving forward or back through the cards, the new card should show the question and not the answer
7. Bonus: Keep score
